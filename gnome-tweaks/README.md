#  GNOME Tweak tool

[Youtube Tutorial - Ubuntu 安裝 GNOME Tweak tool](https://youtu.be/lb_WCmuozcI)

很好用的工具, 更高的自訂化以及設定.

在 ubuntu 中, 預設不會安裝這個 (但 Pop!_OS 預設會裝)

使用以下的指令安裝

```cmd
sudo apt install gnome-tweaks
```

![alt tag](https://i.imgur.com/5MMry7k.png)
